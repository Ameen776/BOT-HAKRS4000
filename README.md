# 🐉 Architect PDF Weaponizer - Telegram Bot

## 📋 الوصف
بوت تيليجرام متكامل لتحويل الصور إلى PDF ملغوم باستخدام تقنية Zero-Width Steganography و BiDi Crash Trap.

## 🚀 النشر على Render

### المتغيرات المطلوبة (Environment Variables)
| Variable | Description |
|----------|-------------|
| `BOT_TOKEN` | توكن البوت من @BotFather |
| `ADMIN_ID` | معرف الشات الخاص بالأدمن |
| `WEBHOOK_URL` | رابط التطبيق على Render |
| `SECRET_KEY` | مفتاح تشفير عشوائي |

### خطوات النشر
1. ارفع الملفات على GitHub
2. أنشئ Web Service جديد على Render
3. اربط المستودع
4. أضف متغيرات البيئة
5. انتظر حتى يكتمل النشر
